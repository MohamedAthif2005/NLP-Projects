# SMS Spam Classifier

A simple NLP project that classifies messages as **spam** or **ham**.

## ✅ What this project does
• Cleans and preprocesses text  
• Converts text to numerical features using TF-IDF  
• Trains a Naive Bayes classifier  
• Predicts whether a message is spam  

## ✅ Technologies used
• Python  
• NLTK  
• Scikit-learn  
• TF-IDF Vectorizer  
• Naive Bayes  

## ✅ Results
The model achieves over **95% accuracy**.

## ✅ How to run
1. Install dependencies  
2. Run the notebook  
3. Test with custom messages
